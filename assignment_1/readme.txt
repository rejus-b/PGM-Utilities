use "make" in the terminal to compile code
use "./pgmEcho [input] [output]" to echo the code
use "./pgmComp [input1] [input2]" to compare two pgm files
use "./pgma2b [input] [output]" to change an input ASCII file to binary
use "./pgmb2a [input] [output]" to change an input binary file to ASCII
use "./pgmReduce [input] [integerFactor] [output]" to reduce a file by an integer reduction factor
use "./pgmTile [input] [integerFactor] [output]" to split an image into nxn smaller images
